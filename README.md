# NutriTrend-AI
Ecosistema analítico avanzado que evalúa el consumo de alimentos en España. Combina Machine Learning (K-Means) para segmentar regiones por riesgo socioeconómico y una arquitectura RAG con la API de Gemini para desplegar un Agente Consultor de Salud Pública. Incluye cuadro de mando interactivo en Power BI para detectar "Hot Spots".
