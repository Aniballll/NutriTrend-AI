# NutriTrend AI 🚀

Ecosistema analítico avanzado que evalúa el impacto de los factores socioeconómicos en el consumo de alimentos en España para la optimización de políticas de salud pública.

## 🛠️ Stack Tecnológico
* **Python & Machine Learning**: Segmentación regional mediante clustering (K-Means) utilizando datos del MAPA e INE.
* **IA Generativa**: Agente Consultor basado en la API de Gemini bajo una arquitectura RAG para emitir recomendaciones ejecutivas sin alucinaciones.
* **Power BI**: Cuadro de mando interactivo para la detección de "Hot Spots" de riesgo nutricional en tiempo real.

## 📂 Estructura del Proyecto
* `analysis.ipynb`: Jupyter Notebook con la ingesta, limpieza de microdata y entrenamiento del modelo.
* `consultor_ai.py`: Script de integración del Agente de IA y lógica de control de flujo de la API.
* `dashboard_nutritrend.pbix`: Informe interactivo con métricas DAX optimizadas.
